What is JsFail?

JsFail is a collection of JavaScript and Json tools glued together from awesome open source projects from around the net. JsFail itself is an open source project written in CoffeeScript and running on Express.

Contributing

Is JsFail missing a tool? Add it! You can find JsFail's GitHub here. The GitHub page also has basic directions on how to get JsFail running on your local computer.
